Program that simulates the package shipping management system for an automated package
shipping operation, in order to practice programming an application with multiple threads
of execution and synchronizing their access to necessary shared objects.

PackageSystem project folder contains: 
-README
-.jar/.java files to run program/ source code
-config.txt
-output.txt

----------------
To run the project from the command line, go to the project folder and
type the following:

java -jar "packageSystem.jar"
